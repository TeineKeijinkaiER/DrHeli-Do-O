# 道央ドクターヘリ 判断支援アプリ（公開・安全版 / DO-O Heli）

機密情報を含まない公開版。スマホからQR/URLで利用できる PWA。

## 収録モード
- 地図モード（**搬送時間と近隣医療機関のみ**。反省・事例等の機微情報は非収録）
- ビギナーモード（運用マニュアル）
- ロジスティックモード（機内チェックリスト・読み上げ）
- クイズモード（4択・検定）

## 公開（GitHub Pages）
1. このフォルダを **独立した公開リポジトリ**として push（例: `DrHeli`）。
2. GitHub → Settings → Pages → Source を「GitHub Actions」に設定。
3. 公開URL（リポジトリ名が `DrHeli` の場合）: `https://teinekeijinkaier.github.io/DrHeli/`

## データ更新
データは非公開リポジトリ（DO-O）側で作成・点検し、**機密を含まない内容のみ**を本リポジトリの `data/` にコピーして push する。
本リポジトリには患者データ・反省記録・編集ツール(admin)は置かない。
